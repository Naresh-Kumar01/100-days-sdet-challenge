# 🎯 Project Objective

The **100 Days SDET Challenge** is a production-style automation portfolio designed to simulate a real enterprise software development lifecycle.

The project demonstrates complete end-to-end implementation of enterprise business scenarios using industry-standard SDET technologies including Java, SQL, JDBC, Selenium, TestNG, Playwright, REST Assured, Appium, Jenkins, Docker, Git, and CI/CD.

Each challenge represents one production-ready business feature from Warehouse Management System (WMS) and Logistics Management System (LMS).
